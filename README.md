# 高爾夫同組競技紀錄 PWA

手機優先的高爾夫同組競技與遊戲紀錄網頁，可離線保存本場資料，支援 1 至 4 人同組計分。

## 功能

- 球員姓名、球場與前九/後九路線設定
- 18 洞總桿與淨桿計分
- 固定差點與新新貝利亞模式
- 逐洞賽零和排行榜
- 奧林匹克推桿遊戲
- 抓蛇遊戲，可設定第 2 至第 4 隻小蛇變大蛇，並支援回上一步
- LocalStorage 本機保存

## 本機執行

```bash
python3 -m http.server 4173
```

開啟：

```text
http://127.0.0.1:4173
```

## 部署

這是純靜態網站，可放在 GitHub Pages、Cloudflare Pages、Netlify、Vercel 或 PageDrop。
